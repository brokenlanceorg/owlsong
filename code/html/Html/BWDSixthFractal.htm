The following fractals were computed in the same manner as the "Newton Fractals" using Newton's method.

Instead using a wave function of one frequency, as in the previous fractals, these images were computed using several frequencies. In fact, these frequencies, when combined, correspond to notes in a D Sixth chord, with the notes and frequencies being the following.

D: 587.329535834815
A: 880
C#: 1108.73052390749
F#: 1479.97769084654

Imagine this fractal existing on an infinite wall and you see the top image in front of you. Assuming that you're roughly 3 feet from the monitor, you could walk backward to "zoom out" to see the bottom image.

However, you would have to walk backward 11,160,000,000 miles to see the second image! That's the distance where most quasars are found--to nearly the edge of the visible universe!

Notice in the first image, there is a light blue spot in the center of the image. This spot is an artefact of losing precision in the computation. Similarly, the washed-out green color of the second image is also a consequence of pushing the utility of the 80 bit 'long double' variables on a PC.
