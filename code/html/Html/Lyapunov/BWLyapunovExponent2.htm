But, for a recursive function, we can’t just consider the first iteration, but all recursions, so that the product of all the absolute values of the derivatives of each recursion is less than one. The number given by equation (4) is known as the Lyapunov exponent for the given function. Since the natural logarithm was taken, the sum is no longer compared to unity, but rather zero. Thus, if the number given in (4) is positive, then the function and the initial seed are ‘unstable’ in that small errors in the initial seed (x0) are magnified. If, however, the Lyapunov exponent is negative, then the function and the initial seed are said to be ‘stable’, that is to say, errors in the initial seed do not infect the calculation in the long run. Observe that the more positive the number, the more unstable the system is, and conversely, the more negative a number, the more stable the system is.
