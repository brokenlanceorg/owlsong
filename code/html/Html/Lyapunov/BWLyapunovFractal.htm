The following images were computed through the use of artificial neural networks.

The network was used as a recursive dynamical system. After a large number of recursions, the sequence of points usually (but not always) converged to a certain value. Thus, in terms of fractal-speak, this image is the attractor of the dynamical system.

In terms of neural networks however, the image is essentially the result of the neurons playing the "association game"; that is, an initial memory is shown to the neurons, then the neurons will remember, or associate, another image given that image. This new memory then becomes the input to the network, which forms yet another new memory, and so on.

The images you see before you are the images that the neurons produced playing this game.
