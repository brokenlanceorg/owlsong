The following images were generated by computing the Lyapunov Exponent for the previous images and then assigning a color to each exponent number and plotting it.

Essentially, the images show how "stable" the dynamical system is. That is to say, given a sequence of points, how far do they drift from the initial value given a small amount of error.

The blue areas signify stable regions in the "brain": the darker the region, the more stable and the brighter the blue, the less stable it is. The green areas signify rather unstable regions in the "brain". The brigher the green, the more choatic that region is in the neural network.

The neural networks used to compute these images were trained to mimic the cellular automaton known as Rule 110, which is known to be a Universal Turing Machine.
