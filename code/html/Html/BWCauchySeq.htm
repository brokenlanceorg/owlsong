[Cauchy Sequence]

Before the definition of a Cauchy Sequence is put forth, a few necessary definitions are in order.

[Definition]

A set X (whose members we generally call points) is said to be a metric space if with any two elements p and q of X there is an associated real number d(p, q) called the distance from p to q such that:

(1) d(p, p) = 0

(2) d(p, q) > 0 when p ¹ q

(3) d(p, q) = d(q, p)

(4) d(p, q) £ d(p, r) + d(r, q)

[Definition]

A sequence {p_n} in a metric space X is said to be a Cauchy Sequence if for every e > 0, there is an integer N such that d(p_n, p_m) < e if n ³ N and m ³ N.
