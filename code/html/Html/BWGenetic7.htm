The ‘evolve()’ method of the abstract class GeneticAlgorithm would look something like the following:

public Individual evolve()
{
Individual theBest = new Individual();
Individual parentOne = null;
Individual parentTwo = null;
Individual child = null;
Vector currPopulationVector = new Vector();

  while( _generationNum++ < _maxGenerationNum ||
    theBest.evaluateFitness() < _wantedFitness )
  {
    while( _curPopulationSize++ < _reqPopulationSize )
    {
      // using selection strategy, get a parent
      // from the pool of Individuals:
      parentOne = theSelectionStrategy.getParent( populationVector );
      parentTwo = theSelectionStrategy.getParent( populationVector );

      child = parentOne.crossover( parentTwo );
      child.mutate();
      currPopulationVector.add( child );
    } // end inner while

    populationVector = currPopulationVector;
    theBest = getBestIndividual();

  } // end outer while

  return theBest;
} // end method evolve
