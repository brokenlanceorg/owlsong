A form of the neural network known as the Hopfield network solves the associative memory problem in a way that most neuroscientists hypothesize the brain stores memories. Probably the neural network’s most salient feature is its ability to generalize knowledge; the neural network can make inferences on data sets that it has never encountered, and if the network has been sufficiently trained, these inferences are usually correct. Knowledge generalization is yet another example of a nearly unsolvable problem in a purely symbolic context.
