Similarly, we use the scalar a to modify the vector’s length. Using these results, we can now define a general backward error propagation algorithm as follows:

(1) Select the training instance, (xi, Tk)
(2) Take the vector xi to be the output from the input layer of neurons
(3) Use equation (III.1) to compute the activations for all j hidden layer neurons
(4) For each hidden unit hj, compute the neuron’s output by equation (III.3)
(5) Compute the output layer’s activation via equation (III.2)
(6) For each output unit ok, compute the neuron’s output by equation (III.4)
(7) Compute E0
(8) Determine DWij and DWjk by equations (IV.1) and (IV.2) respectively
(9) Update weight space by: Wij(t+1) = Wij(t) + DWij, and Wjk(t+1) = Wjk(t) + DWjk.
(10) Repeat steps (1) through (9) for the next training instance, when step (9) is completed for the last training instance, we say that an epoch of training has been completed.
(11) Repeat step (10) for as many epochs as necessary to reduce E0 below some tolerance level.
