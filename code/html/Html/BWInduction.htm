Proposition: The Principle of Induction and the Well-Ordering Principle are logically equivalent.

Before the proof is given, the two principles must be defined and explained.
The Principle of Induction: Given a set S Í N such that 1, 2, ..., k Î S and if n Î S then (n+1) Î S then it must be the case that S = N.

Thus, the principle of induction says that if we have some set that satisfies the conditions of the set S, then it is equivalent to the set of natural numbers.

The Well-Ordering Principle: Every non-empty set of natrual numbers contains a least element.

Suppose we have a subset S Í N, then the well-ordering principle guarrantees that we will always be able to select the smallest element from the set S.

Proof: Consider an arbirary set S Í N and form the set T = {n Î N : n Ï S }. What remains to be shown is that the arbitrary set T has a least element. There are three cases to consider: (1) If 1 Î T, then 1 is the least element of T. (2) If k Î S and (k+1) Ï S, then (k+1) is the least element of T. (3) If k Î S and (k+1) Î S, then by the principle of induction, S = N and therefore T = Æ and since T is empty, it has no element to be the least.

The proof so far has shown that if the principle of induction is true, then the well-ordering principle is true. It remains to show that the well-ordering principle implies the principle of induction.

Define the set A = { n Î N : 1 Î A, " n, when n Î A, (n+1) Î A }, then form the set T = {n Î N : n Ï A }. Obviously, if the principle of induction is true, the set A should be equivalent to the natural numbers and thus the set T will be empty. So, assume that T ¹ Æ. Since T is non-empty, by the well-ordering principle, T must have a least element--call it t. Since t is the least element of T and by the definitions of the sets T and A, it must be the case that (t-1) is an element of A. But, recall that by the definition of the set A, since (t-1) Î A, ((t-1)+1) Î A as well, which is: (t-1+1) = t Î A. This is a contradiction, t has been shown as an element of both of the sets T and A, but by definition the sets are mutually exclusive. Therefore, our only assumption must be false, thus, T = Æ. Therefore, if the well-ordering principle is true, the principle of induction follows as well.
