[Operations]

When dealing with propositions, sets, or objects, it is natural to combine them in different ways. For example, one can take the intersection of two sets, the union of two sets, or the negation of a set. The intersection of two sets A and B (A Ç B) is the set C whose arbitrary element is a member of both sets A and B. The union of two sets A and B (A È B) is the set C whose arbitrary element is a member of either set A or set B, or perhaps even both. The negation of a set is the set of all objects that are not in the respective set.

Fuzzy logic modifies these rules since fuzzy sets aren't amenable to them. Zadeh defined the intersection of two sets A and B, whose membership functions are mA(x) and mB(x) respectively, as the minimum between mA(x) and mB(x). Likewise, the union between two sets A and B is given by the maximum between mA(x) and mB(x). The negation of a set A is defined to be 1 - mA(x).

Of course, there are other ways to perform these operations, some include:

  • Mean: intersection is given by: [ mA(x) + mB(y) ] / 2, while union is given by: [2 * min + 4 * max] / 6, where min = min( mA(x), mB(y) ) and max = max( mA(x), mB(y) ).
  • Mean Squared: same as Mean except the resulting values are squared.
  • Mean(1/n): same as Mean except the resulting values are raised to the 1/nth power.
  • Product: intersection is given by: mA(x) * mB(y) , and union is given by [mA(x) - mB(y)] - [mA(x) * mB(y)].
