This is an image of the complex plane. The bottom and left-most real values are -0.009, while the upper and right-most real values are 0.009. The equation Newton's Method solved for this image is the wave function: f(x) = e^x[ a cos(bx + c) + d sin(ex + g)]. Since this is a close-up of the original image (image 1.), imagine that this fractal image is displayed on a huge wall, and right now, you're standing a couple of feet away from this wall and you see the image before you. Thus, if you walk backward from the wall, you will effectively "zoom out" on the image. If you walked eight-four miles away from the wall, you would then see the first image (image 1.).
