From the client perspective, using the OOGA would look something like the following:

package com.evolution.ga;

SelectionStrategy theSelectionStrat = new RouletteWheelStrategy();
GeneticAlgorithm theGA = new GeneticAlgorithm( theSelectionStrat );
CrossoverStrategy theCross = new SinglePointCrossoverStrategy();

Chromosome theChrome = null;
Individual theIndividual = null;
Vector theGenome = null;

theIndividual = new Individual();
theGenome = new Vector( 0, 1, 0, 1 ); // random genome
theChrome = new BinaryEncodedChromosome( theGenome );
theIndividual.setChromosome( theChrome );
theGA.addIndividual( theIndividual );

// Keep adding Individuals as needed ….

// Now get the best fit individual, or the solution:
theIndividual = theGA.evolve();
