[Fourier Series]

We begin by considering a periodic function of the form:

¦(x) = a sin(bx + c)

An identity from tigonometry tells us that we can write this as:

¦(x) = a[sin(bx)cos(c) + cos(bx)sin(c)]

And if we let A = a cos(c) and B = a sin(c), we can then write this more succinctly as:

¦(x) = A sin(bx) + B cos(bx)

The period of this function is T. For the simple sin(x) function, the period is T = 2p. Since we're multiplying x by b, the period in the above function is T = 2p / b. The period of a periodic function is the distance from one crest to another, one trough to another, or in general wherever the function attains the same value as it did previously.

If we write T = 2L = 2p / b, we can divide out the 2 and then solve for b = p / L

For a non-translated sine or cosine function, any period will have two zeros (two places on the x-axis where the function evaluates to zero). Thus, L in the above equation gives the distance between successive zeros (or half of the period).

Substituting for b in our equation we have:

¦(x) = A sin(xp / L) + B cos(xp / L)

Any periodic function can be written in the above form. Although this is the general case, it will be easier to work with a periodic function of 2p; we can achieve this by substituting T = p / L to get:

¦(x) = A sin(xT) + B cos(xT)
