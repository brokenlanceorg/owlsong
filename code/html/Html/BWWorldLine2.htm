As we know from experiment, light travels at a constant velocity (2.998x108 m/s). To simplify the space-time plane, we scale the plane so that light travels at 45 degree angles, that is to say, for every distance unit it travels, it travels one time unit.

So, if we shine a light in the left direction, it will appear as a 45 degree world-line slanting to the left. If we send two light pulses out both directions, left and right, then the result is known as a light cone, see figure 2.


Figure 2.

All events (points of space-time) that lie within the light cone are events that person A can influence. This is true since nothing can travel faster than light, there is no way for person A to send a signal of any type to events that are outside of A's light cone.

Observe that the horizontal line at t=0 is called the surface of simultaneity. That is, all events that lie on this line occured, in A's frame of reference, at the same time that the light was illuminated.
