As stated earlier, fuzzy logic violates the Law of Excluded Middle and the Law of Non-Contradiction. To see this, consider the crisp logic (classical) notion of the laws. In Boolean logic, the Law of Excluded Middle states that for any set A and its negation set ~A, A Ç ~A = null. In other words, an object or element can not belong to both its own set and the negation of its own set simultaneously.

Looking at the example of the fuzzy set "tall" given above, a person who is 5 feet 7 inches tall (67 inches) is 0.791666 percent "tall"; furthermore, the person is NOT "tall" 1 - 0.791666 = 0.2083333 percent. If we take the Zadeh intersection, we find that this person is min(0.791666, 0.208333) = 0.2083333 percent tall and not tall!

In classical logic, the Law of Non-Contradiction states that for any set A and its complement ~A, the union of A and ~A is the entire universe of discourse U, A È ~A = U.

However, using the example just given, we find that a person who stands 67 inches is tall or not tall max(0.791333, 0.208333) = 0.7916666 percent! Or, using the Mean union operator, we find the answer to be: [ 2(0.208333) + 4(0.791666) ] / 6 = (0.416666 + 3.16666666) / 6 = 0.5972222 percent tall or not tall.

It is these features that give fuzzy logic its power to handle situations that involve misinformation, or ambiguous information.

[Hedges]

Another facet of fuzzy logic that is particularly efficacious, especially in fine-tuning a rule-base, is the hedge. A hedge is simply a modification to an existing fuzzy set--it hedges the value into something a little different. An example is the 'somewhat' hedge; we could modify the above fuzzy set from "tall" to "somewhat tall" to give the set a little more meaning--even if it is more fuzzy than before. So, we could ask the question: "Is someone who stands 67 inches somewhat tall?"
