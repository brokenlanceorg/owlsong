The 'somewhat' hedge is considered to be a member of the class of diluting hedges, that is, hedges that increase the membership function. The 'somewhat' hedge is generally defined on a set A as: somewhat( mA(x) ) = sqrt( mA(x) ). Thus, in our example, we find that someone who stands 67 inches is 'somewhat tall': somewhat( 67 ) = sqrt( 0.7916666 ) = 0.889756 percent, so the hedge serves to aggrandize the underlying fuzzy set.

In general, a diluting hedge is defined as: dilute( mA(x) ) = [mA(x)](1/n), where n is a natural number. There are many other types of hedges including the following.

(1) Restriction Hedges:
  • Below: This hedge is defined on the lower half of a fuzzy set, it reverses the values of the truth function. Defined on a set A with midpoint m as: not( mA(x) ) and an omega-cut defined at m.
  • Above: The converse of the 'Below' hedge.
  • Not: The fuzzy logic negation of a set: not( mA(x) ) = 1 - mA(x).

(2) Concentration and Diluting Hedges:
  • Very, Extremely: These are concentrating hedges and serve to concentrate or focus the fuzzy set making the fuzzy set smaller in mass. Defined as: con( mA(x) ) = mA(x)n, for the 'very' hedge, n = 2, for 'extremely' n = 3. One could define other concentrating hedges such as 'slightly' with n = 1.7 or whatever one chooses.
  • Somewhat, Rather, Quite: These are diluting hedges which perform the converse of the concentrating hedges. The general definition is: dilute( mA(x) ) = mA(x)(1/n), for the 'somewhat' hedge, n = 2, for the 'rather' hedge, n = 3, etc.
(3) Intensification and Diffusion Hedges: The intensification hedge serves to 'intensify' an existing hedge, that is to say, it makes the underlying fuzzy set more crisp, or less fuzzy, while the diffusion hedge makes the underlying fuzzy set more fuzzy.
