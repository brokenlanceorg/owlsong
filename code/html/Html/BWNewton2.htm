Thus an initial 'guess' is made, call it p0, and it is plugged into the iterated system to obtain the value p1; then, this value is entered into the system to obtain the value p2 and so on. This iterated system engenders a sequence {f(p0), f(p1), f(p2), ...} which, for well-behaved functions, converges to zero. When the f(pk) value is smaller than some pre-defined e > 0 we take pk to be the solution or root of the equation f(x).

Geometrically, the value pn from the iterated formula, gives the point on the x-axis where the tangent line to f(pn-1) intersects.

It is well known that if our initial 'guess' p0 is chosen close enough to the solution point, the Newton-Raphson method will always converge to the solution point.
