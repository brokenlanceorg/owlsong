• Composite Maximum: This method finds the greatest value among the consequent sets, then uses that domain point as the crisp value much in the same way the earlier example used the inverse of the truth function.
  • Average Maximum: This method calculates the mean of the upper n points in the set, and then takes the inverse.
  • Near and Far Edges: This method finds either the nearest or the farthest edges of the support set and uses the domain value as the crisp value.
