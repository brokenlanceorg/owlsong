• Positively, Absolutely, Definitely: These intesification hedges are generally defined by: intense( mA(x) ) = mA(x)(1/n), when mA(x) > 0.5, mA(x)n otherwise.
  • Generally, Usually: These diffusion hedges perform the converse of the intensification hedges and are defined as: diff( mA(x) ) = [ mA(x)-0.5]n + 0.5, when mA(x) > 0.5, [ mA(x)+0.5](1/n) - 0.5 otherwise.

[Fuzzy Logic Propositions]

Finally, the full power of fuzzy logic can be seen when one considers the fuzzy logic rules of inference. Generally, we will have a set of rules which operate on a set in input variables and yields a single output variable; we can notate this as: y = R(x1, x2, ..., xn). Notice however, that the output variable is a single crisp number, not a fuzzy set.

At some point during the inference process, we must transform our fuzzy consequent sets into crisp variables, or 'defuzzify' the fuzzy sets.

The fuzzy rule base will consist of several propositions, and each proposition consists of an antecedent and a consequent. For instance, the monotonic proposotion: if x is Y, then z is W consists of the antecedent 'x is Y' and the consequent 'z is W'. The rule base could also consist of unconditional propositions such as: 's is T', but in general, the rule base will have conditional propositions of the form: if (x1 is Y1) Ä (x2 is Y2) Ä...Ä (xn is Yn) then y is Z1, where 'Ä' is either the 'AND' operator or the 'OR' operator.

Looking at the simple case, if x is Y, then z is W, the antecedent will be true to some degree. This value will be used to scale the consequent set W, which will then be defuzzified to yield the output value z. For example, consider the rule: if x is tall, then x's weight is heavy. Here, Y = "tall", while W = "heavy" and z = "x's weight".

To calculate this value, we must define the fuzzy set "heavy". Consider mHeavy(x) = x / 130 - 70 / 130; this defines a linear manifold over the support set [70 lbs, 200 lbs] so that 70 lbs is not heavy and 200 lbs is considered heavy.
