[Zermelo-Fraenkel Axioms]

The following are only some of the axioms from the Zermel-Fraenkel set theory. They are also cast into natural language because had they retained their mathematical character, the simple meaning they embody would be obscured.

(1) Two sets are identical if they have the same members.
(2) The empty set exists.
(3) If X and Y are sets, then the unordered pair {X,Y} is a set.
(4) The union of a set of sets is a set.
(5) Infinite sets exist.

(6) Any property that can be formalized in the language of set theory can be used to define a set.
(7) One can form the power set of a set (i.e., the collection of all subsets of a given set).
(8) The axiom of choice.
(9) X does not belong to X.

Essentially, all of mathematics can be deduced from the ZF formal system of axioms. This is precisely what Bertrand Russell and Alfred North Whitehead proved with their mammoth and momentous Principia Mathematica. Observe the undefined concepts employed here. The notion of a 'set' is undefined, infinity is undefined, members of a set are undefined, the empty set is undefined, and so on. To prevent infinte regress, this must be done.
