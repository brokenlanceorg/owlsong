[Definition]

The weak omega-cut threshold is the value o such that when x ³ o, m(x) = 0. Similarly, the strong omega-cut is the value such that when x > o, m(x) = 0.

[Definition]

The membership function m(x) is a well-defined function [a, b] ® [0, 1], where a and b are real numbers, that describes the truth value for a given support set.

For example, consider the fuzzy set "tall", whose support set will be defined on the interval [4ft, 6ft], or equivalently in inches, [48, 72]. Define m(x) = x / 24 - 2 as the membership function--this defines a linear truth function over the support set. Now we ask: "is a person who stands 5 feet 7 inches a tall person?" Well, to answer, we just input the value into the fuzzy set's membership function: m(5'7'') = m( 67 ) = 67 / 24 - 2 = 2.7916666 - 2 = 0.7916666, of course, this varies with the population, but according to our fuzzy set, this person is roughly 80 percent tall, or this person belongs a little less than 80% to the set of tall people.

The membership function is not limited to simple linear functions, it can be any function desired. Some examples of fuzzy set truth functions include:

  • Linear Manifold: A linear function as described above, m(x) = mx + b.
  • Sigmoid or Logistic: A sigmoid, or S-shaped, curve whose definition can be one of many forms, an easily computable version is: for a £ x £ b, 2[ (x-a)/(g- a) ]², and for b £ x £ g, 1-2[ (x-g)/(g- a) ]², an alpha-cut would be placed at a, and an omega-cut would be placed at g. In the above, a is chosen so that m(a) = 0, g is such that m(g) = 1, and b is such that m(b) = ½.
  • Pi Curves: A 'bell-shaped' curve that also has several different forms depending on the requirements.
  • Beta Curves: An other bell-shaped class of functions that vanish only at inf, an example is: m(x) = 1 / [ 1 + [(x-g)/ b]² ], where g is the center of the bell, and b is the width of the base.
  • Gaussian: A beta curve give by: m(x) = e(-k( g-x)²)
  • Irregular: Any combination of the above. For example, triangular or trapazoidal.
