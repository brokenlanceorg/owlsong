[How is Fuzzy Logic Used?]

Like most artificial intelligence techniques, fuzzy logic redeems itself by solving problems that traditional computing techniques find intractable. Fuzzy logic can be used in much the same way that we reason about the world, so that a fuzzy rule base can be set up that models, to a large degree, how we, as humans, think about a particular problem.

For example, fuzzy logic inference engines have been used in diverse problem spaces such as, cement kilns (the first real comercial application), cameras, thermostats, subway systems, expert systems (soybean pathology, meteorology, financial, etc.)...

II. Theory

As has been stated, fuzzy logic allows members of a set to have varying degrees of membership, namely, the membership of an object takes on the values in the interval [0, 1]. This membership is determined by the fuzzy set, and it is this property that gives fuzzy logic its power.

[Definition]

The support set is the set of values for which the membership function is non-zero.

[Definition]

The weak alpha-cut threshold is the value such that when x £ alpha, the membership function m(x) = 0. The alpha-cut is said to be a strict, or strong, alpha-cut when x < alpha, then m(x) = 0.
